# DetX - Autonomous Landmine Detection System

## Project Overview

DetX is an autonomous landmine detection system leveraging magnetic field sensing technology. Developed as a Year 1, Semester 1 project (Nov 2023 – Mar 2024), this system demonstrates the practical application of robotics and IoT to address critical safety challenges.

## Key Features

- **Magnetic Field Sensing**: Utilizes advanced magnetic sensors to identify metallic landmines
- **4WD Arduino-Based Robot**: Custom-built robotic platform with autonomous navigation capabilities
- **Rotating Sensor Arm**: Optimized coverage area for efficient detection
- **Real-Time Alerts**: Provides immediate feedback upon mine detection
- **Visual Feedback System**: Data visualization techniques for enhanced situational awareness
- **Autonomous Navigation**: Refined algorithms for intelligent path planning and obstacle avoidance
